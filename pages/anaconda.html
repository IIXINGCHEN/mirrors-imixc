anaconda镜像使用帮助

使用说明

由于Anaconda镜像高达3TB+，故我们不进行全量镜像，而是进行缓存。回源站点为清华开源软件镜像站。所有数据被请求过一次后即会缓存到服务器硬盘，缓存30天（对于会变动的文件，缓存时间为2小时）。

Anaconda 是一个用于科学计算的 Python 发行版，支持 Linux, Mac, Windows, 包含了众多流行的科学计算、数据分析的 Python 包。

Anaconda 安装包可以到 http://mirrors.imixc.top/anaconda/archive/ 下载。

修改用户目录下的 .condarc 文件:

channels:
  - defaults
show_channel_urls: true
default_channels:
  - http://mirrors.imixc.top/anaconda/pkgs/main
  - http://mirrors.imixc.top/anaconda/pkgs/free
  - http://mirrors.imixc.top/anaconda/pkgs/r
custom_channels:
  conda-forge: http://mirrors.imixc.top/anaconda/cloud
  msys2: http://mirrors.imixc.top/anaconda/cloud
  bioconda: http://mirrors.imixc.top/anaconda/cloud
  menpo: http://mirrors.imixc.top/anaconda/cloud
  pytorch: http://mirrors.imixc.top/anaconda/cloud
  simpleitk: http://mirrors.imixc.top/anaconda/cloud

即可添加 Anaconda Python 免费仓库。Windows 用户无法直接创建名为 .condarc 的文件，可先执行 conda config --set show_channel_urls yes 生成该文件之后再修改。

运行 conda create -n myenv numpy 测试一下吧。

Miniconda 镜像

Miniconda 是一个 Anaconda 的轻量级替代，默认只包含了 python 和 conda，但是可以通过 pip 和 conda 来安装所需要的包。

Miniconda 安装包可以到 http://mirrors.imixc.top/anaconda/miniconda/ 下载。
